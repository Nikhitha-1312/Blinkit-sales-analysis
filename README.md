# 🛒 Blinkit Sales Analysis Dashboard

## 📊 Overview
This project analyzes Blinkit sales data using Power BI to identify trends, product performance, and outlet insights.

## 🎯 Objectives
- Analyze total sales performance
- Identify top product categories
- Compare outlet performance
- Evaluate fat content impact

## 🧰 Tools Used
- Power BI
- Excel
- SQL
- DAX
- Data Modeling

## 📌 KPIs
- Total Sales
- Average Sales
- Item Count
- Average Rating

## 📈 Dashboard Features
- Sales by Outlet Size
- Sales by Location
- Sales by Item Type
- Fat Content Analysis
- Outlet Trend
- Interactive Filters

## 🗂 Data Model
Star schema with:
- Fact Table: Sales
- Dimension Tables: Items, Outlet, Location

## 💡 Business Insights
- Tier 3 outlets generate highest sales
- Medium outlets perform best
- Regular fat products sell more
- Fruits & snacks are top categories
